<h1 align="center">README-Reader</h1>
<h4 align="center">A GitHub-style tool for opening README files</h4>
<p align="center">
This little tool lets you open your Markdown files — your MDs or READMEs — right on your own computer, completely offline. It shows them cleanly and neatly, just like GitHub, so everything looks organized and nice.</p>
<p align="center">This tool was created purely for reading Markdown files. Personally, I often download projects from GitHub, and when the internet in Iran was cut off or restricted, reading README files and their explanations wasn’t enjoyable for me — I care a lot about style and organization. That’s why I decided to build this tool myself. Its core uses marked.min.js and github-markdown.css, which are well-known projects and played a key role in developing this tool.</p>

<h1 align="center">README-Reader</h1>
<p align="center">ااین ابزار کوچیک بهت اجازه می‌ده فایل‌های Markdown خودت — همون MD یا README — رو مستقیم روی سیستم خودت و کاملاً آفلاین باز کنی. همه چیز رو تمیز و مرتب، درست مثل گیت‌هاب، نشون می‌ده و ظاهرش خیلی جذاب و قابل فهمه</p>
<p align="center">این ابزار صرفاً با هدف خواندن فایل‌های Markdown ساخته شده. من خودم شخصاً معمولاً پروژه‌ها رو از گیت‌هاب دانلود می‌کنم و زمانی که اینترنت در ایران قطع یا محدود بود، خوندن فایل‌های README و توضیحاتشون برام جذاب نبود — من خیلی به نظم و استایل اهمیت می‌دم. بنابراین تصمیم گرفتم خودم این ابزار رو بسازم. هسته این ابزار از marked.min.js و github-markdown.css استفاده می‌کنه که پروژه‌های معروفی هستند و توی توسعه این ابزار نقش اصلی رو داشتن.</p>
